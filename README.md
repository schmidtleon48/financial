# Financial
## Darlehen
Das Script darlehen.py analysiert einen möglichen Fremdkapitalhebel für eine belibige Assetklasse für die Darlehenstypen, Annuität und Tilgung. Man kann schön sehen wie eine hohe Rendite verstärkt wir und eine schwache Rendite weiter gedämpft wird. Abzuwägen bleibt ob das Risko in der jeweiligen Assetklasse gerechtfertigt ist.
